# Ecoleta
 ## Esse é o projeto chamdo "Ecoleta". O objetivo é através da tecnologia fazer a conexão e a comunicação entre estabelecimentos que coletam resíduos com as pessoas que desejam descartar esses resíduos. 
<p align="center">
  <img src="https://github.com/adelarmello/next-level-week/blob/master/assets/logo.png" width="25%">
</p>
<p align="center">
  <img src="https://user-images.githubusercontent.com/30738742/85092498-2755e800-b1c0-11ea-8617-7e46c2abc08e.png" width="40%"></p>
 

Foi desenvolvido um sistema web onde as empresas podem se cadastrar com seu nome, uma foto do estabelecimento, email, whatsapp e definindo um local no mapa onde fica localizada a empresa.

Além disso foi criado um aplicativo móvel onde o usuário primeiramente define um estado e uma cidade ele poderá verificar os pontos de coleta, tambem pode selecionar cada tipo de resíduo que é recolhido naquele local e poderá entrar em contato com a empresa através do email ou whatsapp.
